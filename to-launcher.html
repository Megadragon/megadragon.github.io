Tanki Online Launcher – программа, предназначенная для запуска игры Танки Онлайн через любой установленный браузер, с автоматическим входом на любой сервер и с любыми языковыми настройками.

Итак, что же в моей программе такого особенного? Чтобы ответить на этот вопрос, нужно разобрать понятие лаунчера.

Лаунчеры

Лаунчерами (англ. launch – запускать) называются программные оболочки, основное функциональное предназначение которых состоит в запуске других программ. Различают два вида лаунчеров:

Лаунчеры для запуска разных программ
Лаунчеры данного типа обычно содержат список программ, и при нажатии на конкретный элемент списка запускается связанная с ним программа. К лаунчерам этого типа можно отнести и рабочие столы разных операционных систем.
Лаунчеры для запуска одной программы с разными параметрами
Не секрет, что существуют программы, имеющие множество параметров запуска. Одни из таких параметров просто меняют стиль отображения результатов работы программы, в то время, как другие могут кардинально изменить режим её работы. Такие параметры иначе называются ключами и вводятся с помощью командной строки.
Однако новому поколению пользователей, которые начинали работать с компьютером со времён операционной системы Windows XP или даже более поздних, такой способ запуска приложений не слишком удобен. Поэтому, в частности для удобства таких пользователей, и разрабатываются лаунчеры данного типа.
По организации графического интерфейса пользователя такие лаунчеры подразделяются на два типа:
С примитивным интерфейсом
Такие лаунчеры зачастую ограничиваются двумя текстовыми полями и, иногда, диалоговым окном выбора исполняемого файла, к которому необходимо применить указанные в дальнейшем параметры. Недостаток очевиден: отсутствие контроля правильности ввода параметров и передача строки с ними в том виде, в котором её ввёл пользователь, непосредственно запускаемой программе. Стоит заметить, что некоторые программы из-за неправильного ввода одного из параметров могут не воспринимать в лучшем случае один следующий параметр, а иногда и все остальные.
И чем анализировать, а затем и исправлять, введённую пользователем строку с ключами запуска, гораздо менее затратно её сформировать автоматически на основе выбранных пользователем параметров из списка доступных. Этим и занимаются лаунчеры с продвинутым интерфейсом.
С продвинутым интерфейсом
Эти лаунчеры вместо обычного текстового поля для всех параметров сразу содержат множество таких полей по числу параметров. А в случае, когда список значений того или иного параметра строго ограничен, вместо текстовых полей применяются выпадающие списки, флажки, радиокнопки и прочие элементы графического интерфейса.

Очевидно, что рассматриваемая программа относится ко второму подтипу второго типа: лаунчер для запуска одной программы с разными параметрами и продвинутым интерфейсом.

Особенности

  • Выпадающие списки для каждого из параметров ссылки;
  • Цветовой индикатор кнопок запуска и закрытия программы;
  • Оригинальный подход к оформлению информации об авторе в виде внутриигрового баннера.
